The style guide gives us a set of rules that our website should follow. This was created by my partner, Christian Carrido. We chose two colors that would make the website look clean. The red is used to make things pop out more, while the blue is more subtle.

I took surverys of friends at CPP in order to create a user persona of the target audience we aim to have our website made for. We looked for key features other websites had in order to have inspiration for our own.
